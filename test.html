🖼️ [ Project 2 Image ]

NibbleLog

A full-stack calorie tracking web app built with React, Vite, and Supabase. It features email authentication, real-time meal logging, and a 7-day nutrition history dashboard so users can spot patterns in their eating habits over time. I integrated the Open Food Facts API with debounced search and live barcode scanning to make food lookup seamless across millions of products. On the design side, I built a fully responsive UI using Tailwind CSS and shadcn/ui with a custom design system
